Software Defined Servers
Un centro de datos definido por software es un espacio donde se virtualiza la infraestructura a través de la abstracción, la agrupación de recursos y la automatización mediante el uso de un software inteligente. Un SDDC (centro de datos definido por software) es una instalación de almacenamiento de datos en la que todos los elementos de la infraestructura (redes, almacenamiento, CPU y seguridad) se virtualizan y se entregan como un servicio. El despliegue, la operación, el aprovisionamiento y la configuración se abstraen del hardware. Esas tareas se implementan a través de inteligencia de software. El ex CTO de VMware, Steve Herrod, es ampliamente reconocido por haber acuñado el término alrededor de 2012. El SDDC es el resultado de años de evolución en la virtualización de servidores. Extiende la virtualización de cálculo para almacenamiento y recursos de redes, y proporciona un único conjunto de herramientas de software para administrar esos recursos virtualizados. Además, el SDDC permite la automatización del aprovisionamiento y la gestión basada en políticas, lo que acelera la entrega de recursos y mejora la eficiencia.
