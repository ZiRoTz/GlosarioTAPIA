Cloud Networking
Cloud networking o red de cloud es un tipo de infraestructura donde las funciones y los recursos de red están disponibles a la carta a través de un proveedor de servicios externo que los aloja en una plataforma de cloud. Los recursos de red pueden incluir enrutadores, cortafuegos y ancho de banda virtuales y software de gestión de red; junto con otras herramientas y funciones que se ofrecen a medida que se necesitan. Las empresas pueden usar recursos de red de cloud para gestionar una red interna o utilizar todos los recursos en la cloud. Por «red de cloud» se entiende una red informática definida por software que se encuentra dentro de una infraestructura informática de cloud. Los usuarios y los recursos informáticos internos y externos pueden usar la red para conectarse y comunicarse entre sí. Operar a través de una red en la nube le permite satisfacer las necesidades comerciales con más ancho de banda para manejar los picos de tráfico. Al mover los recursos a una red en la nube, puede optimizar el flujo de tráfico proveniente de cualquier punto de su sistema, ya sea global o en el sitio. Además, debido a la conectividad confiable de las redes en la nube, puede reducir la latencia y aumentar la disponibilidad de las aplicaciones, lo que permite cargas de trabajo de alta intensidad más eficientes.
