White-box servers Original-Design Manufacture
Un servidor de caja blanca se refiere a un servidor de centro de datos creado por fabricantes de diseño original (ODM) mediante el uso de componentes comerciales listos para usar (COTS). Es utilizado principalmente por profesionales de centros de datos que requieren una amplia personalización en el producto. El término caja blanca simplemente significa que el equipo no tiene marca o es genérico. Todas las piezas se compran por separado, lo que ayuda a reducir costos y proporciona a los aficionados y profesionales de centros de datos una mayor personalización para adaptarse mejor a sus necesidades. La facilidad de personalización permite reemplazar partes individuales, en lugar de reemplazar todo el servidor cuando falla el equipo. Los servidores de caja blanca están siendo utilizados cada vez más por los desarrolladores de Open Computer Project (OCP). Los principales sistemas operativos, como Windows Server y Red Hat Enterprise Linux, junto con el software de visualización, se pueden ejecutar en un servidor de caja blanca.
