Software Defined Storage (Almacenamiento definido por software)
El almacenamiento definido por software (SDS) es un sistema de desvinculación del almacenamiento de datos que permite separar el aprovisionamiento y la gestión del almacenamiento del hardware subyacente. El almacenamiento definido por software (SDS) es una arquitectura que separa el software de almacenamiento del hardware. A diferencia de los sistemas tradicionales de almacenamiento adjunto a la red (NAS) o red de área de almacenamiento (SAN), el SDS generalmente está diseñado para ejecutarse en cualquier sistema x86 o estándar del sector; por eso el software no depende de ningún hardware propietario. El almacenamiento definido por software (SDS) permite a usuarios y organizaciones desacoplar o abstraer recursos de almacenamiento de la plataforma de hardware subyacente para obtener más flexibilidad, más eficiencia y una escalabilidad más rápida mediante la programación de los recursos de almacenamiento.
