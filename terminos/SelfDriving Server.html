Self-Driving Server
Un centro de datos autónomo es un centro de datos que está altamente automatizado hasta el punto en que requiere poca intervención humana para su mantenimiento. El término se usa por analogía con los automóviles autónomos que pueden navegar por sí mismos. El término se describe como un objetivo teórico, ya que la mayoría de los centros de datos aún requieren administradores de sistemas. Un centro de datos autónomo se describe como un objetivo final que deben alcanzar los centros de datos del futuro. La construcción de un centro de datos autónomo implica el uso de la automatización y el aprendizaje automático para automatizar muchas tareas de mantenimiento del centro de datos hasta el punto en que los centros de datos puedan funcionar sin intervención humana. El avance hacia la automatización se ha visto favorecido por la aparición del almacenamiento de estado sólido y la virtualización/containerización de servidores. Dado que más servidores se administran con software en lugar de hardware, esto los convierte en un objetivo fácil para la automatización. Un centro de datos autónomo podría ahorrar costos de mano de obra y aumentar la confiabilidad.
